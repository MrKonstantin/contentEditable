jQuery.ContentEditable is a wrapper for modern browsers' DHTML text editing capabilities using the contentEditable=true attribute. It was initially developed for Draft Essay, but I decided to open-source it to further the plugin's development.

Example

Press the "Try it out" button for a demonstration to edit this text. All editable content will have a grey left border, defined by the CSS style .fresheditable.

Usage

  1. Include the jquery.contenteditable.js script.
  2. Call $(".fresheditable").fresheditor(); to initialise
  3. Use $(".fresheditable").fresheditor("edit", true); to start editing.

<script type="text/javascript" src="jquery.contenteditable/jquery.contenteditable.js"></script>

Dependencies

jQuery ContentEditable depends on shortcut.js for its keyboard shortcuts, as given below.

Keyboard Shortcuts

  • Tab indents the selected content
  • Shift+Tab outdents selected content
  • Ctrl+L inserts a link
  • Ctrl+B makes text bold
  • Ctrl+I makes text italic
  • Ctrl+U underlines text
  • Ctrl+Alt+0 formats a paragraph
  • Ctrl+Alt+1 through Ctrl+Alt+5 adds headings 1 through 5
  • Ctrl+J creates an unordered list, like this one
  • Ctrl+K creates an ordered list:
    1. with numbered items...
    2. like this.
